## What is Apprise?
Apprise is a powerful and flexible push notification library that allows you to send notifications to a wide variety of services using a simple, unified interface.

Apprise allows you to send notifications to services like:

- **Slack**
- **Discord**
- **Telegram**
- **Email** (SMTP)
- **Microsoft Teams**
- **Pushover**
- **Gotify**
- And many more!

## Key Features

- **Simple API**: Send notifications with just a few lines of code
- **Multiple Services**: Support for 100+ notification services
- **URL-Based Configuration**: Configure services using simple URLs
- **Flexible**: Works in Python, as a CLI tool, or via API
