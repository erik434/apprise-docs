    <div class="apprise-why-content">
      <h2>Why Apprise</h2>

      <p>One notification library to rule them all.</p>

      <ul>
        <li><strong>Universal:</strong> A single, intuitive URL format for many notification providers.</li>
        <li><strong>Rich Content:</strong> Attachments and images supported where the upstream service permits.</li>
        <li><strong>Lightweight:</strong> Designed for automation workflows with minimal overhead.</li>
        <li><strong>Responsive:</strong> Amazing response times because all messages sent asynchronously.</li>
        <li><strong>Flexible:</strong> Works as a library, a CLI tool, or a self-hosted API service.</li>
      </ul>
